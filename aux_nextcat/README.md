# aux_nextcat — 다음카테고리 예측 데이터셋

전처리 버전 관리 항목(과거 작업 백업).

## 구성
- **종류**: 다음카테고리 예측 데이터셋
- **X 구성**: nextcat_dataset(.fullwin).npz
- **Y 구성**: 다음 카테고리 멀티클래스
- **재현 소스**: `scripts/nextcat_1735.py, save_full_windows.py` (원본 위치 유지)
- **원본 데이터**: `sample_project/data/processed_nextcat` (16.5 MB)
- **백업**: `output/` (복사됨) — 상세 `output/DATASET_BACKUP.md`

> 대용량은 git 비대화 방지를 위해 기본 포인터. `--all`로 물리 복사 가능.
